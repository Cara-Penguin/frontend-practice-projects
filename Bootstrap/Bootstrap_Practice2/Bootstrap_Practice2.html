偏移的區塊

可以通過.offset- Class來位移欄 。網格位移的空間與欄間格相符合，這種推移對於快速排版可增加更多變化。

大區塊(.col-sm-4.col-md-7.col-lg-6)

Bootstrap 的網格系統預設會自動適應六個斷點以及您自定義的任何斷點。(xs、sm、md、lg、xl、xxl)

尺寸是向上覆蓋的(比如如果指定md中螢幕佔7欄，沒有指定lg大螢幕佔多少，會沿用中螢幕的分配)
